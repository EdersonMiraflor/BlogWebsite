OJT Tech Stack

Express JS

During my OJT, I used Express JS to build the backend for this blog, creating RESTful APIs for data management. This enhanced my skills in server-side development and middleware, contributing to the blog’s efficient content handling and my growth in scalable web solutions.

Flutter

I explored Flutter during OJT to develop cross-platform mobile apps, learning widget-based design and state management. This influenced the blog’s responsive layout, reflecting my ability to adapt web design principles from mobile development.

Angular

Angular was key to my OJT frontend work, used to build this blog’s interactive UI with two-way data binding. It powered the input forms and dynamic content, showcasing my proficiency in modern web frameworks.

Supabase

I integrated Supabase during OJT for the blog’s data storage and image uploads, streamlining backend tasks. This demonstrated my ability to leverage cloud solutions, enhancing the blog’s data management capabilities.

GitHub was vital for OJT collaboration, managing this blog’s codebase and tracking changes with mentors. It highlighted my skills in version control and teamwork, ensuring a professional development process.

Android Studio
